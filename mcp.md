您在Cline中安装了5个MCP服务器，以下是详细清单：

## 📋 已安装的MCP服务器清单

### 1. **Software Planning MCP**

`github.com/NightTrek/Software-planning-mcp`

**用途**：软件开发规划和任务管理

- 启动规划会话，设定开发目标
- 保存实施计划
- 添加、删除和管理待办事项（todos）
- 更新任务完成状态
- 跟踪项目复杂度和进度

**适用场景**：项目规划、任务分解、开发流程管理

---

### 2. **Playwright MCP**

`github.com/microsoft/playwright-mcp`

**用途**：浏览器自动化和网页测试

- 浏览器导航和操作（点击、输入、滚动等）
- 网页截图和快照
- 表单填写和文件上传
- 网络请求监控
- 控制台日志收集
- 运行Playwright代码片段

**适用场景**：Web应用测试、UI自动化、网页抓取、端到端测试

---

### 3. **Design to Code MCP**

`github.com/jeremylongshore/claude-code-plugins-plus-skills/.../design-to-code`

**用途**：设计稿转代码

- 将设计图转换为可用的代码
- 辅助UI实现

**适用场景**：前端开发、设计稿实现

---

### 4. **Context7 MCP**

`github.com/upstash/context7-mcp`

**用途**：编程库文档查询和代码示例检索

- 解析库/包名称为Context7兼容的库ID
- 查询最新的编程库文档
- 获取代码示例和最佳实践
- 支持各种编程框架和库（如MongoDB、Next.js、Supabase等）

**适用场景**：学习新技术、查找API文档、获取代码示例

---

### 5. **21st.dev Magic MCP** ✨ (刚安装)

`github.com/21st-dev/magic-mcp`

**用途**：AI驱动的UI组件生成

- 通过自然语言描述创建UI组件（使用 `/ui` 或 `/21` 命令）
- 搜索和获取专业品牌Logo（JSX/TSX/SVG格式）
- 浏览和获取组件灵感
- 改进现有组件（添加动画和高级功能）
- 集成SVGL品牌资源库

**适用场景**：快速原型开发、UI组件创建、品牌资源获取

---

## 💡 使用建议

所有服务器当前都是**启用状态**（disabled: false），您可以随时使用它们的功能：

- **开发规划时**：使用 Software Planning MCP
- **需要测试网页时**：使用 Playwright MCP
- **需要创建UI组件时**：使用 Magic MCP（输入 `/ui` 命令）
- **查找技术文档时**：使用 Context7 MCP
- **设计转代码时**：使用 Design to Code MCP

这些工具可以大大提升您的开发效率！
